### Installation instructions
```
curl -ksLO https://go.dev/dl/go1.21.3.linux-amd64.tar.gz
tar xfz go1.21.3.linux-amd64.tar.gz
export GOROOT=$PWD/go
export PATH=$PATH:$GOROOT/bin
go version
go doc http
```
